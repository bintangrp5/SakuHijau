# 🌱 Saku Hijau – Sistem Manajemen Bank Sampah

## 📌 Deskripsi

Saku Hijau adalah aplikasi desktop yang dikembangkan menggunakan **Java** dan **MySQL** untuk membantu digitalisasi pengelolaan bank sampah. Aplikasi ini dirancang untuk menggantikan pencatatan manual menjadi sistem yang lebih terstruktur, efisien, dan mudah dikelola.

## 🎯 Fitur Utama

* 👤 **Manajemen Nasabah**
  Mengelola data nasabah (tambah, lihat, ubah, hapus)

* ♻️ **Transaksi Sampah**
  Mencatat setoran sampah, proses penimbangan, dan perhitungan nilai

* 📊 **Laporan Otomatis**
  Membuat laporan transaksi dan aktivitas secara cepat dan sistematis

* 🖥️ **Antarmuka Sederhana**
  Tampilan desktop yang mudah digunakan untuk operasional sehari-hari

## 🛠️ Teknologi yang Digunakan

* **Java** (Aplikasi Desktop)
* **MySQL** (Database)
* **JDBC** (Koneksi Database)

## 🧠 Konsep yang Diterapkan

* CRUD (Create, Read, Update, Delete)
* Manajemen Database Relasional (RDBMS)
* Desain User Interface (UI) Desktop

## 🚀 Tujuan

Aplikasi ini dibuat untuk:

* Meningkatkan efisiensi administrasi bank sampah
* Mengurangi kesalahan pencatatan manual
* Mendukung pengelolaan sampah yang lebih sistematis
* Meningkatkan kesadaran terhadap lingkungan

## ⚙️ Cara Instalasi

1. Clone repository ini:

   ```bash
   git clone https://github.com/username/saku-hijau.git
   ```

2. Buka project di IDE Java (NetBeans / IntelliJ / Eclipse)

3. Setup database MySQL:

   * Buat database (contoh: `saku_hijau`)
   * Import file `.sql` yang tersedia

4. Konfigurasi koneksi database di project

5. Jalankan aplikasi
